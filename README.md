# Student-Performance-Insight-Dashboard-Power BI
 Power BI dashboard for analyzing student performance, pass/fail rates, and study factors.
 
 Student Performance Insight Dashboard (Power BI)
📌 Overview

This project is a Power BI dashboard that analyzes student performance data. It helps to understand how different factors like study hours, environment, and motivation affect student results.

🎯 Objectives
Analyze pass and fail percentage
Study impact of study environment
Understand relationship between study hours and scores
Compare performance across grades

Dataset
The dataset contains 1000 student records with the following features:

Study hours
Attendance
Final score
Motivation score
Study environment
Parent education level

📊 Dashboard Insights
Around 93% students passed and 7% failed
Students in quiet environments perform better
Higher study hours and motivation lead to higher scores
Grade A students have highest average scores

🛠 Tools Used
Power BI
Data Visualization
DAX
Basic Data Analysis
